After recieving some feedback, and also noting some of my personal areas for improving the website, I went forth and made some new changes!

This includes new and improved button styling, a more browser-safe (and pretty cool) font, and some backend changes to improve the site's scalability and modularity for future changes.

As you can also see, I have switched from Sublime Text to Visual Studio Code with my own custom theme! I took the Dark High Contrast theme and expanded upon it with my own syntactic highlight changes, and implemented OhMyPosh Bash terminal for Git Control.

I've also been working on my various projects lately, including a quick project I worked on last night where I build the QPI (Quick Project Initialiser) Tool. QPI was a solution to a problem I faced wherein I found I was spending time setting up project directories and settings that I could have automated so I can get straight into the coding.

The tool allows for CLI usage by typing "qpi" in the desired directory location, or specifying via "qpi -b 'path/to/project'" or "qpi --build 'path/to/project'". You can also check the version using "qpi -v" or "qpi --version", and to init a project for python use "qpi -py" or use the "-py" flag when running the build.

The first version of it is up on my GitHub. There is currently only general structure support and python specific support, but I plan to expand the support to other languages, and to support virtual environments and full git setup.

That is all for now, looking forward to my next projects. :>
